# Smart Waste Sorter ♻️

An intelligent waste classification system leveraging **machine learning** and **computer vision** to automate the sorting of waste into categories like biodegradable, non-biodegradable, and recyclable. Designed for environmental impact, this project showcases how AI can be used to support sustainability in real-world applications.

---

## 🚀 Features

- Image classification model trained to detect multiple waste categories
- Real-time prediction using webcam or image input
- Accuracy optimization using preprocessing techniques via OpenCV
- Clean and modular code for easy extension and deployment
- *(Optional)* IoT integration for automated sorting via servo motors
- *(Optional)* Web API via Flask/FastAPI for deployment
